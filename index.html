Calendar

Виджет календаря для интеграции в страницу. Управляется клавиатурой, либо мышью. Демо:

Фишки

  • Управление клавиатурой
  • Чистый javascript (без зависимостей)
  • Aвтокомплит месяца

Установка

Подключите все содержимое скаченного архива к сайту.

Использование

Чтобы открыть виджет нажмите на , а чтобы его срыть кликните по любой части документа. Управление виджетом, делится на две категории:

Виждет закрыт(виджет в фокусе) Виджет открыт
Hotkey
      up/down - увеличивает/уменьшает текущий год/месяц/день
      tab/left/right - изменяют позицию курсора
      Если вводим месяц:
          up/down - предыдущая/следующая подсказка месяца
          right/space/tab - вставить месяц
      up/down/left/right - изменение текущего дня/месяца/года
      alt + left/right - переход на следующий месяц/год/года
      alt + up - подняться на уровень выше
      alt + down - опуститься на уровень ниже
